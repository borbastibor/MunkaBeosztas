# MunkaBeosztas

<strong>Feladat leírása:</strong>
Szükség volt egy weboldalra, amellyel dolgozókat, gépkocsikat és feladatokat adott dátumon össze lehet rendelni, úgynevezett kiküldetések formájában és ezen adatokat naptár formában meg lehet jeleníteni, valamint felületet biztosítani az adatok szerkesztésére, törlésére, hozzáadására (CRUD).

<br />
<strong>Megvalósítás:</strong>
A weboldal létrehozásához a CodeIgniter 3 keretrendszer került felhasználásra az egyszerű telepíthetősége, konfigurálhatósága és használata miatt. Az oldal kialakításánál a Model-View-Controller tervezési minta lett alkalmazva, mivel egy elég komplex és szerteágazó feladatot kellett megoldani adatbázis eléréssel és a nézetek nagy száma miatt. Az adatbázismotor kiválasztásánál szempont volt a kis méret és a hordozhatóság, illetve a könnyű kezelhetőség, így az SqLite 3-ra esett a választás.
