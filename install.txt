FÜGGŐSÉGEK

- Windows IIS vagy Apache webserver
- PHP 7.1-es verzió vagy újabb
- Composer - PHP függvénykönyvtár kezelő
- PhpSpreadsheet függvénykönyvtár

------------------------------------------------------------------------------------

TELEPÍTÉS

Windows (Apache webserver):
    1. XAMPP letöltése és telepítése. (elég csak a webserver opció választása)
       https://www.apachefriends.org/hu/index.html

    2. Composer alkalmazás letöltése és telepítése. (PHP függvénykönyvtárak kezeléséhez)
       https://getcomposer.org/

    3. PhpSpreadsheet telepítése a parancssorból a Composer használatával.
       ("composer require phpoffice/phpspreadsheet")
       FONTOS!!! A Composer létre fog hozni egy "Vendor" könyvtárat a megadott helyre.
       Ezt a könyvtárat majd be kell másolni a weblap forrásfáljait tartalmazó
       könyvtárba, amely az apache webservernél a "..\xampp\htdocs\weblapkönyvtár".

    4. A weblap forrásfájljainak a bemásolása a "..\xampp\htdocs\weblapkönyvtár"-ba.

Windows (Windows IIS):
    1. IIS engedélyezése a Windows szolgáltatásoknál.

    2. A Windows Web Platform Installer letöltése és telepítése.

    3. PHP környezet telepítése a Web Platform Installer-el.

    4. Composer alkalmazás letöltése és telepítése. (PHP függvénykönyvtárak kezeléséhez)
       https://getcomposer.org/

    5. PhpSpreadsheet telepítése a parancssorból a Composer használatával.
       ("composer require phpoffice/phpspreadsheet")
       FONTOS!!! A Composer létre fog hozni egy "Vendor" könyvtárat a megadott helyre.
       Ezt a könyvtárat majd be kell másolni a weblap forrásfáljait tartalmazó
       könyvtárba, amely az apache webservernél a "..\inetpub\wwwroot\weblapkönyvtár".

    6. A weblap forrásfájljainak a bemásolása a "..\inetpub\wwwroot\weblapkönyvtár"-ba.

------------------------------------------------------------------------------------

FUTTATÁS

Windows (apache webserver):
    1. Indítsuk el az XAMPP controlpanel-t.

    2. Az XAMPP alkalmazásban indítsuk el az apache webserver-t. (az XAMPP-t ne zárjuk
       be ezután)

    3. Nyissunk egy webböngészőt és írjuk be a webserver címét és a weblapot
       tartalmazó könyvtár nevét (pl.: http://localhost/weblapkönyvtár). A weblap
       automatikusan el fog indulni.

Windows (Windows IIS):
    1. Indítsuk el a windows szolgálatatásoknál az IIS-t (alapbeállítások használatánál
       a szolgálatatás automatikusan elindul a Windows indulásakor).

    2. Nyissunk egy webböngészőt és írjuk be a webserver címét és a weblapot
       tartalmazó könyvtár nevét (pl.: http://localhost/weblapkönyvtár). A weblap
       automatikusan el fog indulni.
